# 프로젝트 배경 및 컨텍스트

## 토지거래허가제(토허제)란?

토지거래허가제는 투기 방지를 위해 특정 지역에서 일정 면적 이상의 토지를 거래할 때
시장/군수/구청장의 허가를 받아야 하는 제도입니다.

### 서울시 토허제 타임라인

- **2024.10.15**: 서울 전역 토지거래허가구역 지정 ("10·15 대책")
- 이전에는 강남, 서초, 송파, 용산 등 일부 지역만 지정

## 데이터 출처

### 서울시 부동산정보광장

- URL: https://land.seoul.go.kr
- 토지거래허가 신청/처리 현황 제공
- API 제한: 한 번에 62일까지만 조회 가능

### 데이터 수집 범위

- **기간**: 2025.10.15 ~ 현재
- **지역**: 서울시 25개 자치구 전체
- **갱신**: 매일 09:00 (KST) 자동 수집

## 처리결과 구분

| 상태 | 설명 |
|------|------|
| 허가 | 토지거래 승인 |
| 취소 | 허가 후 취소 |
| 취하 | 신청자가 신청 철회 |
| 불허가 | 허가 거부 |
| 반려 | 서류 미비 등으로 반려 |
| 미결 | 심사 진행 중 |

## 건물명 매핑 로직

1. 지번 주소로 카카오 주소 검색 API 호출
2. 도로명 주소의 `building_name` 필드 추출
3. 매핑 결과는 `building_name_map.json`에 캐시
4. 잘못된 매핑은 `manual_building_names.json`에서 오버라이드

### 수동 오버라이드 예시

```json
{
  "노원구 월계동 85-7": "서울원아이파크",
  "노원구 하계동 284": "하계동 한신청구아파트"
}
```

## 신규 데이터 표시

- `수집일자` 필드로 신규 여부 판단
- 가장 최근 수집일자와 일치하면 "신규" 뱃지 표시
- 상단에 별도 "신규 데이터" 섹션으로 강조

## 알려진 제한사항

1. **API 데이터 누락**: 서울시 API가 과거 데이터를 일부만 제공
   - 기사 기준 2024.10~12월 약 9,935건
   - API 조회 시 약 1,000건만 반환

2. **건물명 정확도**: 카카오 API가 정확한 건물명을 반환하지 않는 경우 있음
   - 수동 오버라이드로 보정 필요

3. **실시간성**: 하루 1회 갱신으로 실시간 데이터 아님
